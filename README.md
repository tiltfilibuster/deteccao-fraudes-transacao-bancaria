# detecção fraudes em transações bancárias
Um experimento de machine learning treinando modelos para detectar fraudes em transações bancárias e avaliando os pontos fortes e fracos de cada modelo. Assim como avaliação de métodos de análise de dados para avaliar os modelos

###requisitos:
-pandas
-numpy
-scikit-learn
-imbalanced-learn
-xgboost
-matplotlib
-shap
